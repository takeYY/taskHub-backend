# TaskHub-Backend

## 環境構築

必要なパッケージのインストール

```bash
npm i
```

## 起動方法

- プログラムの起動

  ```bash
  # development
  npm run start

  # watch mode
  npm run start:dev

  # production mode
  npm run start:prod
  ```

- ESLint による構文チェックと整形

  ```bash
  npm run lint
  ```

- Firebase エミュレータの起動

  Firestore, Functions のエミュレータが起動し、本番環境の DB が汚染されない。エミュレータ環境の DB は空っぽの状態で起動し、起動毎にリセットされる。

  ```bash
  npm run emulators
  ```

- Function エミュレータ起動

  Functions のエミュレータが起動し、本番環境の DB と接続する。

  ```bash
  npm run fb
  ```

## 新たなデータ追加

`<DATA>`は適切に置き換える

```bash
# src/<DATA>/<DATA>.module.tsの作成
nest g module <DATA>
# src/<DATA>/<DATA>.service.tsの作成
nest g service <DATA>
# src/<DATA>/<DATA>.resolver.tsの作成
nest g resolver <DATA>
```

## デプロイ

```bash
npm run deploy
```
